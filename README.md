Title
Mall Customer Segmentation using K-Means Clustering
Overview
This project uses unsupervised machine learning to segment mall customers based on their purchasing behavior.
Technologies Used
Python
Pandas
Matplotlib
Seaborn
Scikit-learn
Steps
1. Data exploration
2. Feature scaling
3. Elbow method
4. K-Means clustering
5. Cluster visualization
Result
The model identified five distinct customer segments that can help businesses improve marketing strategies.
